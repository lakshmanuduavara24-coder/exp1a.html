Working of Hyperlinks in HTML

A hyperlink is created using the <a> (anchor) tag. The href attribute specifies the destination URL (where the link goes). The target attribute specifies where the linked document will open.

Example 1: Basic Hyperlink

Visit Google

In this example, the href attribute directs the browser to open Google. Since no target attribute is used, the link opens in the same tab.

Example 2: Open Link in New Tab

Visit Wikipedia (Opens in New Tab)

Here, the target="_blank" attribute opens the link in a new tab.

Common target Attribute Values

  • _self - Opens the link in the same tab (default).
  • _blank - Opens the link in a new tab.
  • _parent - Opens the link in the parent frame.
  • _top - Opens the link in the full body of the window.
